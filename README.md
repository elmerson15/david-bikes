# David's Bikes

A modern Flutter + Firebase workshop management application built for bike service centers and repair shops.

---

## Features

- Admin Login Authentication
- Vehicle Management
- Customer Details Storage
- Invoice Creation
- PDF Bill Generation
- Recent Bills Tracking
- Firebase Firestore Database
- Tamil Font Support
- Search Vehicle Functionality

---

## Tech Stack

- Flutter
- Firebase Authentication
- Cloud Firestore
- PDF Package
- Dart

---

## Screens

### Login Screen
Admin authentication system.

### Dashboard
Main control panel for workshop management.

### Add Vehicle
Store customer and vehicle details.

### Vehicle List
View and manage all registered vehicles.

### Invoice System
Generate service invoices and bills.

### PDF Generation
Create printable bills with logo and watermark.

## Firebase Setup

This project uses Firebase.

To run locally, add your own:

- `google-services.json`
- `GoogleService-Info.plist`
- `firebase_options.dart`

These files are excluded from the public repository for security reasons.

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/david-bikes.git
cd david-bikes

```
